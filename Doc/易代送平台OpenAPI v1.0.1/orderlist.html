物流订单接收接口

系统参数

请参照API简介

应用参数

应用参数主体如下:

参数名称 类型 是否必须 描述
order_id string 是 订单号 第三方的订单号，一个第三方用户同一个订单号只能推送一次
create_time date 是 订单下单时间
receive_time date 是 要求送餐时间
is_pay boolean 是 用户是否已付款，true 已付款 false 未付款
payment int 是 支付类型 0 现金
total_price number 是 订单金额，精确到两位小数
address address 是 收货地址信息
store_info store 是 门店信息
order_details orderdetail[] 是 订单明细
remark string 否 备注
weight number 否 重量，默认为0
delivery_fee number 否 外送费,默认为0
package_count int 否 订单数量，默认为1（并单时使用)

address-收货地址信息

参数名称 类型 是否必须 描述
user_name string 是 用户姓名
user_phone string 是 用户联系电话
province_code string 否 用户所在省份code
city_code string 否 用户所在城市code
area_code string 否 用户所在区域code
province string 是 用户所在省份名称
city string 是 用户所在城市名称
area string 是 用户所在区域名称
address address 是 用户收货地址
longitude number 否 用户收货地址所在区域经度
latitude number 否 用户收货地址所在区域纬度
pickupcode string 否 取货码

store-门店信息

参数名称 类型 是否必须 描述
store_id int 是 对接方店铺ID
store_name string 是 店铺名称
id_card string 否 店铺身份证号
phone string 是 门店联系电话
phone2 string 否 门店第二联系电话
address string 是 门店地址
province_code string 否 门店所在省份code
city_code string 否 门店所在城市code
area_code string 否 门店所在区域code
province string 是 门店所在省份名称
city string 是 门店所在城市名称
area string 是 门店所在区域名称
longitude number 否 门店所在区域经度
latitude number 否 门店所在区域纬度
delivery_fee number 否 外送费,默认为0
commission_type int 否 佣金类型，涉及到快递员的佣金计算方式，默认1

orderdetail-订单明细信息

参数名称 类型 是否必须 描述
detail_id string 是 明细ID，默认为0 （启用，并且不能重复，一个订单明细只能推送一次
detail_id number 是 商品名称
unit_price number 是 商品单价，精确到两位小数
quantity int 是 商品数量
